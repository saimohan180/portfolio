# Zero-Trust Portfolio (GitHub Pages)

Modern static portfolio website with smooth animations and a cybersecurity-inspired visual style.

## Files

- `index.html` - page structure and content
- `styles.css` - responsive styles, theme, and animations
- `script.js` - interactions (typing terminal, reveal effects, counters, filters, tilt)

## Deploy on GitHub Pages

1. Create a GitHub repository and push these files.
2. Open repository settings.
3. Go to **Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main` (or your default branch)
   - Folder: `/ (root)`
5. Save and wait for deployment.
6. Your live URL will appear in the Pages section.

## Customize

- Update project cards in `index.html`.
- Replace the Formspree action URL with your own form endpoint.
- Tune colors and glow intensity in `:root` variables in `styles.css`.

## Notes

- No build step required.
- Optimized for desktop and mobile.
- Includes reduced-motion support for accessibility.
